# P1308 统计单词数 题解

## 题目大意

给定一个单词和一篇文章，要求统计这个单词在文章中出现了多少次，并输出它第一次出现的位置。

需要注意：

1. 匹配时不区分大小写；
2. 必须是完整单词匹配；
3. 如果没有出现，输出 `-1`；
4. 位置从 `0` 开始计算。

例如：

```text
To
to be or not to be is a question
```

其中 `To` 和 `to` 不区分大小写，所以文章中有两个独立的 `to`，第一次出现的位置是 `0`，输出：

```
2 0
```

------

## 解题思路

这道题的关键是判断一个单词是不是在文章中作为“独立单词”出现。

比如要查找：

```
to
```

在文章中：

```
Did the Ottoman Empire lose its power at that time
```

虽然 `Ottoman` 里面看起来包含 `to`，但是它不是一个独立单词，所以不能算。

因此我采用的思路是：

> 把文章按照空格分成一个个单词，再逐个和目标单词比较。

------

## 处理大小写

题目要求不区分大小写，所以先把目标单词和文章都转换成小写。

例如：

```
To
```

转换后变成：

```
to
```

转换代码：

```
for (int i = 0; i < w.size(); i++) {
    if (w[i] >= 'A' && w[i] <= 'Z') {
        w[i] = w[i] - 'A' + 'a';
    }
}
```

文章也用同样的方法处理。

------

## 按单词扫描文章

使用一个变量 `j` 从左到右扫描文章。

如果遇到空格，就跳过。

如果遇到非空格字符，说明一个新单词开始了，记录它的起始位置：

```
int start = j;
```

然后继续向后扫描，直到遇到空格为止，把这个单词取出来：

```
string temp = "";

while (j < s.size() && s[j] != ' ') {
    temp += s[j];
    j++;
}
```

此时 `temp` 就是文章中的一个完整单词。

然后判断：

```
if (temp == w)
```

如果相等，说明找到了一次。

------

## 记录答案

使用两个变量：

```
int count = 0;
int first = -1;
```

其中：

- `count` 表示目标单词出现的次数；
- `first` 表示第一次出现的位置；
- `first = -1` 表示还没有找到过。

每当找到一次：

```
count++;
```

如果是第一次找到，就记录位置：

```
if (first == -1) {
    first = start;
}
```

------

## 完整代码

```
#include <iostream>
#include <string>
using namespace std;

int main() {
    string w;
    string s;

    getline(cin, w);
    getline(cin, s);

    // 把目标单词转换成小写
    for (int i = 0; i < w.size(); i++) {
        if (w[i] >= 'A' && w[i] <= 'Z') {
            w[i] = w[i] - 'A' + 'a';
        }
    }

    // 把文章转换成小写
    for (int i = 0; i < s.size(); i++) {
        if (s[i] >= 'A' && s[i] <= 'Z') {
            s[i] = s[i] - 'A' + 'a';
        }
    }

    int count = 0;
    int first = -1;

    int j = 0;

    // 从左到右扫描文章
    while (j < s.size()) {
        // 跳过空格
        if (s[j] == ' ') {
            j++;
            continue;
        }

        // 当前单词的起始位置
        int start = j;

        // 取出当前单词
        string temp = "";

        while (j < s.size() && s[j] != ' ') {
            temp += s[j];
            j++;
        }

        // 判断当前单词是否等于目标单词
        if (temp == w) {
            count++;

            // 第一次出现时记录位置
            if (first == -1) {
                first = start;
            }
        }
    }

    if (count == 0) {
        cout << -1 << endl;
    } else {
        cout << count << " " << first << endl;
    }

    return 0;
}
```

------

## 样例解释

输入：

```
To
to be or not to be is a question
```

先全部转换成小写：

```
to
to be or not to be is a question
```

文章中的单词依次是：

```
to
be
or
not
to
be
is
a
question
```

其中 `to` 出现了 2 次。

第一次出现的位置是 `0`。

所以输出：

```
2 0
```

------

## 总结

这道题可以用“按空格切分单词”的方法解决。

核心步骤：

1. 读入目标单词和文章；
2. 把它们都转换成小写；
3. 从左到右扫描文章，取出每一个完整单词；
4. 判断是否等于目标单词；
5. 统计出现次数，并记录第一次出现的位置。
