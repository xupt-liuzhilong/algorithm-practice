# P2010 回文日期 题解

## 题目大意

给定两个日期 `date1` 和 `date2`，日期格式为：

```text
YYYYMMDD
```

要求统计在 `date1` 到 `date2` 之间，包括 `date1` 和 `date2` 本身，有多少个日期是回文日期。

所谓回文日期，就是这个 8 位数字从左往右读和从右往左读完全一样。

例如：

```
20111102
```

反过来还是：

```
20111102
```

所以它是回文日期。

## 方法一：逐日枚举日期

### 思路说明

一种比较直接的思路是：

1. 从起始日期 `date1` 开始；
2. 判断当前日期是不是回文；
3. 如果是，答案加一；
4. 日期往后加一天；
5. 一直枚举到 `date2` 为止。

这种做法比较符合直觉，重点是需要能够正确处理日期加一天。

例如：

```
20110101
20111231
```

我们可以从 `2011-01-01` 开始，一天一天往后枚举，直到 `2011-12-31`。

## 判断回文

日期可以用字符串表示，例如：

```
20111102
```

判断它是不是回文，只需要使用双指针：

```
bool IsPalindrome(string date) {
    int left = 0;
    int right = date.size() - 1;

    while (left < right) {
        if (date[left] != date[right]) {
            return false;
        }

        left++;
        right--;
    }

    return true;
}
```

如果左右两边对应字符全部相同，就说明是回文日期。

------

## 日期加一天

为了方便处理日期，可以写一个简单的 `Date` 类，保存年月日。

日期加一天时，要注意：

1. 每个月天数不同；
2. 2 月要判断闰年；
3. 月份超过 12 时，年份要加一。

判断某年某月有多少天：

```
int GetMonthDay(int year, int month) {
    int monthDays[13] = {
        0, 31, 28, 31, 30, 31, 30,
        31, 31, 30, 31, 30, 31
    };

    if (month == 2 && ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0)) {
        return 29;
    }

    return monthDays[month];
}
```

闰年的判断规则是：

```
能被 4 整除但不能被 100 整除，或者能被 400 整除
```

## 完整代码：逐日枚举

```
#include <iostream>
#include <string>
using namespace std;

// 判断字符串形式的日期是否是回文
bool IsPalindrome(string date) {
    int left = 0;
    int right = date.size() - 1;

    while (left < right) {
        if (date[left] != date[right]) {
            return false;
        }

        left++;
        right--;
    }

    return true;
}

class Date {
public:
    Date(int year, int month, int day) {
        _year = year;
        _month = month;
        _day = day;
    }

    // 获取某年某月的天数
    int GetMonthDay(int year, int month) {
        int monthDays[13] = {
            0, 31, 28, 31, 30, 31, 30,
            31, 31, 30, 31, 30, 31
        };

        if (month == 2 && ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0)) {
            return 29;
        }

        return monthDays[month];
    }

    // 日期加 day 天
    Date& operator+=(int day) {
        _day += day;

        while (_day > GetMonthDay(_year, _month)) {
            _day -= GetMonthDay(_year, _month);
            _month++;

            if (_month > 12) {
                _year++;
                _month = 1;
            }
        }

        return *this;
    }

    // 转换成 YYYYMMDD 格式的字符串
    string ToString() {
        string s = "";

        s += char(_year / 1000 + '0');
        s += char(_year / 100 % 10 + '0');
        s += char(_year / 10 % 10 + '0');
        s += char(_year % 10 + '0');

        s += char(_month / 10 + '0');
        s += char(_month % 10 + '0');

        s += char(_day / 10 + '0');
        s += char(_day % 10 + '0');

        return s;
    }

private:
    int _year;
    int _month;
    int _day;
};

int main() {
    string date1;
    string date2;

    cin >> date1 >> date2;

    int year = (date1[0] - '0') * 1000
             + (date1[1] - '0') * 100
             + (date1[2] - '0') * 10
             + (date1[3] - '0');

    int month = (date1[4] - '0') * 10
              + (date1[5] - '0');

    int day = (date1[6] - '0') * 10
            + (date1[7] - '0');

    Date d(year, month, day);

    int count = 0;

    // 日期格式是 YYYYMMDD，所以可以直接用字符串比较大小
    while (date1 <= date2) {
        if (IsPalindrome(date1)) {
            count++;
        }

        d += 1;
        date1 = d.ToString();
    }

    cout << count << endl;

    return 0;
}
```

## 方法一易错点

### 1. 日期字符串要及时更新

如果只写：

```
d += 1;
```

但是没有把日期对象重新转成字符串：

```
date1 = d.ToString();
```

那么 `date1` 一直不会变化，循环可能会死循环。

### 2. 终止日期也要统计

题目要求统计区间 `[date1, date2]`，包括两个端点。

所以循环条件应该写：

```
while (date1 <= date2)
```

不能写成：

```
while (date1 != date2)
```

否则会漏掉终止日期。

### 3. 大数组或复杂类不是必要的

这题数据范围并不大，逐日枚举可以理解日期处理过程，但代码相对复杂。

# 方法二：枚举年份构造回文日期

## 新思路说明

上面的做法是一天天枚举日期，思路直观，但比较麻烦。

这道题还有一个更巧妙的方法：

> 一个 8 位回文日期的后 4 位，其实由前 4 位年份唯一决定。

日期格式是：

```
YYYYMMDD
```

如果它是回文，那么后四位 `MMDD` 必须等于年份 `YYYY` 的反转。

例如年份是：

```
2011
```

那么反过来就是：

```
1102
```

所以对应的回文日期只能是：

```
20111102
```

也就是：

```
2011 年 11 月 02 日
```

因此我们不需要枚举每一天，只需要枚举年份。

------

## 构造月份和日期

假设：

```
int year = 2011;
```

年份的各位数字为：

```
2 0 1 1
```

回文日期应该是：

```
20111102
```

所以：

```
month = 11
day = 02
```

可以这样写：

```
int month = year % 10 * 10 + year / 10 % 10;
int day = year / 100 % 10 * 10 + year / 1000;
```

------

## 判断日期是否合法

不是每个年份反转后都能得到合法日期。

例如：

```
2019
```

构造出的日期是：

```
20199102
```

月份是 `91`，显然不合法。

所以还需要判断：

1. 月份是否在 `1 ~ 12`；
2. 日期是否在当前月份允许的范围内；
3. 2 月要考虑闰年。

------

## 完整代码：枚举年份

```
#include <iostream>
using namespace std;

// 判断是否是闰年
bool isLeap(int year) {
    return (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
}

// 判断日期是否合法
bool isValidDate(int year, int month, int day) {
    int monthDay[13] = {
        0, 31, 28, 31, 30, 31, 30,
        31, 31, 30, 31, 30, 31
    };

    if (month < 1 || month > 12) {
        return false;
    }

    if (isLeap(year)) {
        monthDay[2] = 29;
    }

    if (day < 1 || day > monthDay[month]) {
        return false;
    }

    return true;
}

int main() {
    int date1, date2;
    cin >> date1 >> date2;

    int ans = 0;

    int startYear = date1 / 10000;
    int endYear = date2 / 10000;

    for (int year = startYear; year <= endYear; year++) {
        // 根据年份构造回文日期
        int month = year % 10 * 10 + year / 10 % 10;
        int day = year / 100 % 10 * 10 + year / 1000;

        // 拼成完整日期 YYYYMMDD
        int date = year * 10000 + month * 100 + day;

        if (isValidDate(year, month, day) && date >= date1 && date <= date2) {
            ans++;
        }
    }

    cout << ans << endl;

    return 0;
}
```

------

## 两种方法对比

| 方法     | 思路                   | 优点                 | 缺点                       |
| -------- | ---------------------- | -------------------- | -------------------------- |
| 逐日枚举 | 从起始日期开始每天加一 | 直观，适合练习日期类 | 代码较长，需要处理日期进位 |
| 枚举年份 | 用年份反推出回文日期   | 简洁高效             | 需要观察到回文日期规律     |

------

## 总结

这道题可以先用逐日枚举理解：

```
日期加一天 + 判断回文
```

但是更推荐掌握枚举年份的做法：

```
YYYYMMDD 是回文
=> MMDD 由 YYYY 反转得到
=> 枚举年份即可
```

核心步骤：

1. 枚举年份；
2. 根据年份反推出月份和日期；
3. 判断日期是否合法；
4. 判断是否在输入范围内；
5. 统计答案。
