# P1014 Cantor 表 题解

## 题目大意

给定一个整数`N` ,要求输出 Cantor表中的第`N`项对应的分数。

Cantor 表的前几项如下：

![image-20260514204456207](C:\Users\liuzhilong\AppData\Roaming\Typora\typora-user-images\image-20260514204456207.png)

按照题目中的Z字顺序排列为：

```
1/1, 1/2, 2/1, 3/1, 2/2, 1/3, 1/4, ...
```

---

## 解题思路

观察 Cantor 可以发现，它是按照一条一条斜线排列的。

第 1 条斜线有 1 个数：

```
1/1
```

第 2 条斜线有 2 个数：

```
1/2  2/1
```

第 3 条斜线有 3 个数：

```
3/1  2/2  1/3
```

第 4 条斜线有 4 个数：

```
1/4  2/3  3/2  4/1
```

所以可以先判断第 `N` 项在哪一条斜线上。

设当前斜线编号为 `line`，每条斜线上的元素个数也等于 `line`。

我们不断用 `N` 减去当前斜线的长度：

```
while (n > line) {
    n -= line;
    line++;
}
```

循环结束后：

- `line` 表示第 `N` 项所在的斜线编号；

- `n` 表示它是这条斜线上的第几个数。

---

## 斜线方向规律

继续观察可以发现：

奇数斜线方向如下：

```
第 3 条斜线：3/1  2/2  1/3
```

分子从大到小，分母从小到大。

所以奇数斜线：

```
分子 = line - n + 1
分母 = n
```

偶数斜线方向如下：

```
第 4 条斜线：1/4  2/3  3/2  4/1
```

分子从小到大，分母从大到小。

所以偶数斜线：

```
分子 = n
分母 = line - n + 1
```

---

## 代码实现

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int line = 1;

    // 找到第 n 项所在的斜线
    while (n > line) {
        n -= line;
        line++;
    }

    int a, b;

    if (line % 2 == 1) {
        // 奇数斜线：分子从大到小，分母从小到大
        a = line - n + 1;
        b = n;
    } else {
        // 偶数斜线：分子从小到大，分母从大到小
        a = n;
        b = line - n + 1;
    }

    cout << a << "/" << b << endl;

    return 0;
}
```

## 样例解释

输入：

```
7
```

前 7 项分别是：

```
1/1
1/2
2/1
3/1
2/2
1/3
1/4
```

所以第 7 项是：

```
1/4
```

输出：

```
1/4
```

## 总结

这道题的核心是找规律。

解题步骤：

1. 先确定第 `N` 项在哪一条斜线上；
2. 再确定它是这条斜线上的第几个数；
3. 最后根据斜线编号的奇偶性判断分子和分母。

时间复杂度为 `O(√N)`，可以通过本题数据范围。
