# P1008 三连击 解题思路

## 题目大意

将数字 `1~9` 分成三组，组成三个三位数，使这三个三位数满足比例：

```text
第一个数 : 第二个数 : 第三个数 = 1 : 2 : 3
```

要求输出所有满足条件的三个三位数。

---

## 解题思路

因为三个数的比例是 `1 : 2 : 3`，所以只需要枚举第一个三位数 `a`。

那么另外两个数就可以直接表示为：

```cpp
b = 2 * a;
c = 3 * a;
```

只要判断 `a、b、c` 这三个数的 9 个数字是否刚好由 `1~9` 组成，并且每个数字只出现一次即可。

---

## 枚举范围

第一个数 `a` 是三位数，所以：

```text
a >= 100
```

第三个数 `c = 3 * a` 也必须是三位数，所以：

```text
3 * a <= 999
```

也就是：

```text
a <= 333
```

所以可以枚举：

```cpp
for (int a = 100; a <= 333; a++)
```

当然，由于不能出现数字 `0`，实际有效答案会自动被判断函数筛掉。

---

## 判断方法

用一个数组 `cnt[10]` 统计数字出现次数。

依次取出 `a、b、c` 的每一位数字：

1. 如果出现数字 `0`，说明不符合条件。
2. 如果某个数字重复出现，也不符合条件。
3. 最后检查 `1~9` 是否每个数字都出现了一次。

## 总结

这道题是典型的暴力枚举题。

核心思路是：

```text
枚举第一个三位数 a，再判断 a、2a、3a 是否刚好使用了 1~9 每个数字一次。
```

这类题不需要复杂算法，关键是把判断条件写清楚。
